Detailed Description

Eye specific data.

Field Documentation

bool Calibrated

Indicates whether the eye is calibrated. If this is false then the GazePoint is not valid for the eye.

bool Found

Indicates whether the eye is found in the image. If this is false then the rest of the data for the eye is not valid.

Examples:
/src/QuickStart/DisplayVideo.cpp.
QLXYPairFloat GazePoint

The position of the gaze point in percentage of the calibrated area. This value is not constrained to the calibrated area.

QLXYPairFloat Glint0

The position in pixels of the image of the center of one of the glints. Compare the x value to determine if this is the left of right glint.

Examples:
/src/QuickStart/DisplayVideo.cpp.

QLXYPairFloat Pupil

The position in pixels of the image of the center of the pupil.

Examples:

float PupilDiameter

The diameter of the pupil in millimeters. The resolution of this measurement is .02mm. If the eye is found and the pupil diameter is less than zero then the diameter could not be determined and it's value should not be used for that frame.
