
to clone repo:
'git clone https://github.com/asucapstone/eyeglass.git'

to add files to a commit:
'git add [/path/to/filename] ... '

to see what is going to be committed:
'git status -s'

to see differences in a file:
'git diff'

before commit, you might have to configure your git:
git config --global user.name 'Your Name'
git config --global user.email you@somedomain.com

to commit, with message
git commit -m 'my hola mundo changes'



to commit all entire project:
git commit -am "message to changes "
